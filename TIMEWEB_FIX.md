# Решение проблемы "This service has been suspended by its owner" на Timeweb Cloud

## 🔴 Проблема: Сервис приостановлен

Если видишь сообщение "This service has been suspended by its owner", это означает что:
- Приложение было остановлено вручную
- Или упало из-за ошибок при запуске
- Или не хватает средств на балансе

## ✅ Решение: Запустить/Перезапустить приложение

### ШАГ 1: Проверь баланс и статус

1. Зайди в **Timeweb Cloud Dashboard**
2. Проверь **баланс** (должно быть достаточно средств для тарифа)
3. Найди свой проект `telegram-quiz` или `Brainy Horologium`

### ШАГ 2: Запусти приложение

**Если приложение остановлено:**

1. В Timeweb Cloud Dashboard найди свой проект
2. Нажми кнопку **"Start"** или **"Resume"** (если есть)
3. Или нажми **"Restart"** / **"Redeploy"**

**Если приложение не запускается:**

1. Нажми **"Redeploy"** или **"Rebuild"**
2. Дождись завершения сборки

### ШАГ 3: Проверь переменные окружения

### ШАГ 4: Проверь логи

1. В Timeweb Cloud Dashboard открой раздел **"Logs"**
2. Посмотри последние логи - там должны быть ошибки если что-то не так
3. Типичные ошибки:
   - Отсутствует `RAILS_MASTER_KEY` → добавь переменную
   - Отсутствует `TELEGRAM_BOT_TOKEN` → добавь переменную
   - Приложение не запускается → проверь логи

### ШАГ 5: Если ничего не помогает - пересоздай приложение

**Если приложение не запускается после всех попыток:**

1. В Timeweb Cloud Dashboard удали старое приложение (или останови его)
2. Создай новое приложение:
   - **Тип:** Docker
   - **Репозиторий:** `Razmik-Kutinava/telegram_quiz`
   - **Ветка:** `main`
   - **Регион:** Москва (MSK-1)
   - **Конфигурация:** 1 CPU, 2 GB RAM, 30 GB NVMe
   - **Health Check Path:** `/up`
   - **Переменные окружения:** добавь все из таблицы выше
3. Нажми **"Create"** или **"Deploy"**

## 🔍 Проверка после запуска

После того как приложение запустится:

1. **Health check:** 
   ```
   https://razmik-kutinava-telegram-quiz-d64a.twc1.net/up
   ```
   Должен вернуть: `{"status":"ok"}`

2. **Главная страница:**
   ```
   https://razmik-kutinava-telegram-quiz-d64a.twc1.net/
   ```
   Должна открыться страница

3. **Проверка webhook:**
   - Отправь `/start` боту в Telegram
   - Должна появиться кнопка "Пройти квиз"

## 📝 Что нужно сделать в Timeweb Cloud Dashboard (пошагово):

1. **Открой Timeweb Cloud Dashboard**
2. **Найди свой проект** (может называться `telegram-quiz` или `Brainy Horologium`)
3. **Проверь статус:**
   - Если статус "Stopped" или "Suspended" → нажми **"Start"**
   - Если статус "Running" но не работает → нажми **"Restart"**
4. **Проверь переменные окружения:**
   - Settings → Environment Variables
   - Убедись что все переменные из таблицы выше установлены
5. **Проверь логи:**
   - Открой раздел "Logs"
   - Посмотри последние записи - там должны быть ошибки если что-то не так

## ⚠️ Важно:

- **Баланс:** Убедись что на балансе достаточно средств (минимум 810 ₽/месяц для тарифа 2 GB RAM)
- **Переменные:** Все переменные окружения должны быть установлены ДО запуска приложения
- **Перезапуск:** После добавления переменных обязательно перезапусти приложение

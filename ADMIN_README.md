# Админ-панель

## Доступ

1. Откройте в браузере: `http://localhost:3000/admin`
2. Введите пароль (по умолчанию: `admin123`)
3. Пароль можно изменить через переменную окружения `ADMIN_PASSWORD`

## Возможности

### Дашборд (`/admin`)
- Общая статистика (пользователи, сессии, сезоны)
- Популярные результаты
- Последние сессии

### Пользователи (`/admin/users`)
- Просмотр всех пользователей
- Поиск по имени, username или ID
- Редактирование данных пользователя
- Удаление пользователя
- Просмотр всех сессий пользователя

### Сессии квиза (`/admin/quiz_sessions`)
- Просмотр всех сессий
- Фильтрация по сезону
- Поиск по пользователю или результату
- Редактирование сессии
- Удаление сессии
- Просмотр ответов пользователя

### Сезоны (`/admin/seasons`)
- Создание новых сезонов
- Редактирование сезонов
- Активация/деактивация сезонов
- Просмотр всех сессий сезона
- Удаление сезонов

## Безопасность

- Пароль хранится в переменной окружения `ADMIN_PASSWORD`
- По умолчанию: `admin123` (измените в production!)
- Сессии хранятся в cookies
- Все админ-страницы защищены авторизацией

## Изменение пароля

Добавьте в `.env` файл:
```
ADMIN_PASSWORD=ваш_новый_пароль
```

Или установите переменную окружения перед запуском:
```bash
export ADMIN_PASSWORD=ваш_новый_пароль
bin/rails server
```
